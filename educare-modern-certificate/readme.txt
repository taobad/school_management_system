Educare modern certificate add-ons.

## == Description ==

Transform certificates effortlessly using our Modern Certificate Template, designed for a sleek and professional appearance.

## == Installation ==

1. Download `Educare Modern Certificate` from official website of FixBD (https://fixbd.net)
2. Then, in your WordPress administration panel, go to Plugins > Add New and click the Upload Plugin button at the top of the page. (For manualy installation, Upload `educare-modern-certificate.zip` to the `/wp-content/plugins/` directory.)
3. Activate the plugin (Educare Modern certificate) through the 'Plugins' menu in WordPress.
4. Make sure Educare premium version is active.
5. Then, Go to "Educare > Settings > Certificate for active and configure the plugin.
